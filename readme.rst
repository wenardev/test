Tes Junior Programmer proyek Wendi Abdul Rohim 

Instalasi
--------------

1. Clone repositori ini: `git clone https://github.com/wenardev/test.git`
2. Upload database yang berada di folder "DB" dengan nama "test.sql"
3. Done.

Penggunaan
-------------

1. Pastikan telah melakukan instalasi proyek seperti yang dijelaskan di bagian Instalasi.
2. Pindahkan direktori proyek ke direktori yang di inginkan.
3. Buka XAMPP/Laragon/atau yang lainnya, dan nyalakan Apache dan MySQL.
4. Buka browser dan akses proyek melalui `http://localhost/test`
5. Untuk mengakses dashboard `http://localhost/test/dashboard`
